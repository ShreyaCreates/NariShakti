# NariShakti – Women Empowerment Campaign Platform

## Project Overview

**NariShakti** is a web-based platform designed to promote women empowerment by allowing users to explore, join, and support various social campaigns. The website provides an interactive space where individuals can register, participate in meaningful initiatives, and contribute through awareness and donations.

---

## Features

* **User Registration & Login**
  Users can create an account, log in securely, and access personalized features.

* **Campaign Browsing & Joining**
  Users can view different women empowerment campaigns and join the ones they support.

* **User Profile Dashboard**
  Displays all campaigns joined by the user in one place.

* **QR Code for Donation (Demo)**
  After joining a campaign, a QR code is generated as a demo for donation purposes.

* **Logout Functionality**
  Users can safely log out of their accounts.

---

## Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL

---

## Project Structure

* `index.php` – Homepage
* `register.php` – User registration
* `login.php` – User login
* `profile.php` – User dashboard
* `campaigns.php` – Campaign listing & joining
* `logout.php` – Logout functionality
* `style.css` – Styling
* `script.js` – Client-side logic
* `database.sql` – Database structure

---

## Setup Instructions

1. Install a local server like XAMPP/WAMP.
2. Place the project folder inside the `htdocs` directory.
3. Import the `database.sql` file into MySQL using phpMyAdmin.
4. Update database credentials in your PHP files (if required).
5. Start Apache & MySQL from the control panel.
6. Open browser and run:

   ```
   http://localhost/WebDev_Project/index.php
   ```
